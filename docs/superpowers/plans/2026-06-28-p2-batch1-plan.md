### Task 1: soul_memory.py core methods

**File:** `plastic_promise/memory/soul_memory.py`

Core methods to implement:
1. **MemoryWorthCalculator.calculate_worth()** — Wilson lower bound
2. **MemoryWorthCalculator.update_counters()** — increment per feedback type
3. **RecMem.__init__()** — store engine reference, init empty defaultdicts for records
4. **RecMem.store()** — create MemoryRecord → engine.store_memory() → embedder.embed() → return record
5. **RecMem.recall()** — embedder.embed(query) → engine.supply(text,vec,type,scope) → return ContextPack
6. **RecMem.update()** — engine.update_memory(id, **updates), handle re-embed if content changed
7. **RecMem.forget()** — engine.delete_memory(id)
8. **RecMem.stats()** — parse engine.memory_stats_json() → dict
9. **RecMem.list_records()** — engine.list_memories(**filters) → list
10. **RecMem.apply_feedback()** — engine.get_memory(id) → record_adopted/rejected → engine.store_memory(record)

Each method must: have try/except, call Rust engine methods, return proper types.

Verify:

